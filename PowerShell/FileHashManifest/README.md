# FileHashManifest.ps1

Скрипт для автоматического вычисления контрольных сумм файлов по заданному списку расширений и сохранения результатов в отдельные `.hash`-файлы.
Ищет файлы указанных форматов в текущем каталоге, вычисляет для каждого файла хеши `MD5`, `SHA1` и `SHA256`, а затем создаёт рядом текстовый файл с тем же базовым именем и расширением `.hash`.



## Использование

1. Помести его в каталог с файлами, для которых нужно вычислить хеши, либо перейди в нужный каталог через PowerShell.
2. При необходимости измени список расширений в переменной `$extensions`.
3. Запусти скрипт.

Пример запуска:

```powershell
.\FileHashManifest.ps1
```

## Сценарии использования
- Подготовка контрольных сумм для архивов и дистрибутивов.
- Проверка целостности файлов после копирования.
- Передача файлов вместе с контрольными суммами.
- Хранение хешей для последующей ручной сверки.

## Ограничения
- Скрипт по умолчанию работает только в текущем каталоге.
- Подкаталоги не обрабатываются, если отдельно не добавить рекурсивный обход.
- При наличии нескольких файлов с одинаковым базовым именем, но разными расширениями, итоговые `.hash`-файлы могут перезаписываться.

Пример конфликта:
- `file.zip` → `file.hash`
- `file.rar` → `file.hash`

В таком случае рекомендуется изменить схему именования итогового файла, например включать полное имя с расширением.
